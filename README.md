# pharma-sales-eda
## Проект №1: Анализ структуры сбыта и сезонности фармацевтических препаратов (EDA)

### Цель: Оптимизация складских остатков на основе анализа сезонности.
### Методы: Использование скользящего среднего для сглаживания тренда.
### Вывод: «Продажи категории X растут на 40% в марте, рекомендуется увеличить закупки в январе».
